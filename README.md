# atviii-atlantis

## Pré-requisitos

- Node.js v18.x ou superior  
  [Baixe aqui](https://nodejs.org/)
- TypeScript instalado globalmente  
  ```bash
  npm install -g typescript

### Instalação das dependências

```bash
npm install
```

### Compilação do TypeScript

```bash
tsc
```

### Execução do sistema

```bash
node src/js/app/app.js
```
